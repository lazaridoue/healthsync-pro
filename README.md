# HealthSync Pro

HealthSync Pro is a distributed health data ingestion system designed to process high volumes of simulated IoT health metrics such as heart rate, sleep cycles, and step counts.

Features:
- FastAPI backend for high-performance ingestion
- Simulated IoT health device data generator
- Stability monitoring for abnormal health patterns
- PyTest unit tests with high coverage
- Real-time health dashboard (planned)

Tech Stack:
- Python
- FastAPI
- PyTest
- React (future)
